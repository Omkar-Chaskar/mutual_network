# Instructions to test the app
```
> Step 1 : Insert Person Name , Relationship type , Friend Name
> Step 2 : Press ADD button to Add data into table.
> Step 3 : Repeat Step 1 and 2 for more than 3 times to get more data to check Degree of Separation.
> Step 4 : Navigate to Separation Page by clicking Seperation button.
> Step 5 : Enter first person name and second person name who we have to find connections  or degree of separation between them.
> Step 6 : Click on Search button to get Degree of Separation between first and second person.
```
